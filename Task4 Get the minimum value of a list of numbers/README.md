# Task 4  Get the minimum value of a list of numbers

Make a function that gets the minimum value of a list of numbers.

## Features  

- Finds and returns the **minimum value** from the list.  

## Example Input  
File content:  
`[7, 2, -1, -5, 5, 4]`  

## Example Output  
`-5`
